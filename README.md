# rvs

## Background

How to use same function by writing different code in react, vue and svelte.

## Start

```shell
npm run dev // base app
cd react
npm run dev // start react micro app
cd vue
npm run dev // start vue micro app
cd svelte
npm run dev // start svelte micro app
```
